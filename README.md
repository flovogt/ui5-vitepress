## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hschaefer123/ui5-vitepress.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start dev server using
   ```js
   npm run dev;
   ```
